TESS (Terminology Expert Supporting System)

TESS is terminology consolidation system by experts. The TESS is composed of three pages; selecting a non-resolved terminology, choosing a standard terminology from a list of candidates and retrieving a consolidated terminology from ULMS.

Experts can carry out the terminology consolidate according to following procedure.

  1. Select an item to be consolidated by clicking a resolving button. The item has 3 columns such as a non-resolved term, an effect sentence and a database name having the non-resolved term.
  2. Check an effect sentence having non-resolved term and select consolidated terminology if there is a standard term in list of candidates.
  3. When there is not a proper consolidated terminology, retrieve a standard terminology from the ULMS.
  4. 4. If a search result still do not has a proper consolidated terminology, expert can add a new consolidated terminology by clicking ‘add a new terminology to dictionary’
